<h1>ChunkEntities</h1>

## Commands :

* /entites
  * Displays a map of chunks with the number of entities
* /entites list
  * Displays the list of entities on the chunk with their number (chunkentities.admin)

## Permissions :

* chunkentites.entite
  * Permission for command /entites
* chunkentities.admin
  * Permission for command /entites list
