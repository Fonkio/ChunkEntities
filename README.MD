<h1>ChunkEntities</h1>

## Commands :

* /entite
  * Displays a map of chunks with the number of entities
* /entite list
  * Displays the list of entities on the chunk with their number (chunkentities.admin)

## Permissions :

* chunkentites.entite
  * Permission for command /entite
* chunkentities.admin
  * Permission for command /entite list
